# daily_stock_analysis — 项目架构

> LLM驱动的多市场股票智能分析系统（A股 + 港股 + 美股）

---

## 一、架构概览

```
┌─────────────────────────────────────────────────────┐
│                   定时触发器 / CLI                     │
└──────────┬──────────────────────────┬──────────────┘
           │                          │
           ▼                          ▼
┌─────────────────────┐   ┌─────────────────────────┐
│  数据获取层 (Fetchers) │   │     LLM 分析引擎         │
│  ┌─────┬─────┬─────┐ │   │  ┌──────────────────┐  │
```

## 二、数据源

| 市场 | 数据源 | 备注 |
|------|--------|------|
| A股 | akshare | 已有基础，覆盖行情/财报/板块 |
| 港股 | yfinance / akshare | yfinance 免费，覆盖面广 |
| 美股 | yfinance | 免费，数据完整 |

## 三、核心流程

**每日运行流程：**
1. 读取配置的股票列表（按市场分组）
2. 并行获取各市场行情数据
3. 技术面分析（均线、MACD、RSI、成交量异常）
4. 基本面关键指标抓取（A股用akshare，港美股用yfinance）
5. LLM 汇总分析 → 生成简报
6. 推送/输出

## 四、LLM 驱动策略

- **日常简报**：将行情数据 + 技术指标输入 LLM，生成每日市场解读
- **信号预警**：当出现异常波动/突破关键均线/量价背离，LLM 分析原因并给出判断
- **深度分析**：对特定股票进行多维度分析（财务 + 技术 + 行业对比）
- **多市场联动**：LLM 对比 A/港/美股相关标的的联动关系

## 五、输出格式

- **飞书推送**：每日市场简报（早上开盘前）
- **Markdown 报告**：保存到 reports/output/
- 根据市场状态区分：交易日推送、非交易日跳过
